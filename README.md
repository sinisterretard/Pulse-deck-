# PULSE DECK ARENA

Original Roblox hero-deck arena shooter MVP built as a Rojo project.

## Prerequisites

1. Install Roblox Studio:
   - https://create.roblox.com/landing

2. Install Rojo:
   - https://rojo.space
   - Recommended: `cargo install rojo` or download from GitHub releases.

## Project Structure

- `default.project.json` (Rojo config)
- `src/` (game files)

## Run with Rojo

From the project root:

```
rojo serve
```

## Open in Roblox Studio

1. Open Roblox Studio.
2. Create a new Baseplate.
3. Install the Rojo plugin if you don't already have it.
4. Click **Connect** in the Rojo plugin.
5. The project will sync into Studio.

## Test the Game

1. Press **Play** in Studio.
2. The Neon Foundry map should generate automatically.
3. Click **PLAY** in the generated UI.
4. Confirm a 5-hero deck, or keep the default deck.
5. Use WASD to move, left mouse to fire, R to reload, Q for ability binding, 1-5 to switch heroes, and V to toggle camera.
6. In solo testing, the Blue bot team is created automatically.
7. Studio/admin debug commands: `/pda_reset`, `/pda_start`, `/pda_bots`, `/pda_winred`, `/pda_winblue`.

## Notes

- Stages 1-6 are now implemented.
- The map is fully generated by code.
- Progression uses DataStore when available and falls back to in-memory session data.
- No external assets or Toolbox content are used.
